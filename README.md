# GSM 학사 AI 가이드

## 프로젝트 목표
- 학생의 학년과 학과를 기준으로 개인화된 학사 정보를 제공한다.
- GSM의 복잡한 규정을 쉽게 설명하고, 학사 일정과 추천 활동을 연결한다.
- AI 챗봇 기반 인터페이스를 통해 학사 상담 경험을 제공한다.

## 폴더 구조
- docs/: 요구사항, 아키텍처, API, DB, 배포 설계 문서
- src/features/chat/: 챗봇 UI
- src/features/guidance/: 개인화 학사 가이드 UI
- src/features/regulations/: 규정 해설 UI
- src/shared/: 공통 API/상수 모듈

## 개발 시 주의사항
- 백엔드와 프론트엔드 역할 분리를 먼저 명확히 한다.
- AI 응답 생성 로직은 서버 측에서 처리하는 구조를 염두에 둔다.
- 개인정보가 포함될 수 있는 입력값은 마스킹 처리 규칙을 먼저 정의한다.
